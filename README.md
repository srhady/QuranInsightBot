## ✨ Features (বৈশিষ্ট্যসমূহ)
* **🤖 Fully Automated:** Runs seamlessly via GitHub Actions (cron jobs). No local server needed!
* **🧠 AI-Powered Research:** Uses Gemini AI to write highly informative posts based on 20+ dynamic Islamic categories (Cosmology, History of Prophets, Tafsir, Science & Quran).
* **📖 Authentic References:** Includes correct Arabic text (with Harakat), reliable Tafsir (e.g., Tafsir Ibn Kathir), and precise historical contexts.
* **🎨 Premium Formatting:** Utilizes Telegram's HTML parse mode for beautiful layouts (Blockquotes, Italics, Bolds) without Markdown errors.
* **⚡ Zero Cost:** Built entirely using free-tier tools (Google Gemini Free API & GitHub Actions).

## 🛠️ Technologies Used (প্রযুক্তিসমূহ)
* **Python 3** (Requests, Random, OS)
* **Google Gemini API** (For dynamic content generation)
* **Telegram Bot API** (For auto-posting to the channel)
* **GitHub Actions** (For scheduling and automation)

## 🚀 How It Works (কীভাবে কাজ করে)
1. The script runs automatically at scheduled times via GitHub Actions.
2. It randomly selects a deep Islamic topic (e.g., "Embryology in the Quran" or "The History of Prophet Nuh").
3. It passes a strict, logic-based prompt to the Gemini API.
4. The AI generates a 500-word premium post featuring Arabic verses, blockquote translations, and logical analysis.
5. The script pushes the formatted text directly to the Telegram channel using the Bot Token.
